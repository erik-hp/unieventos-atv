# UniEventos

Aplicativo Flutter para catálogo de eventos acadêmicos da universidade.

## Funcionalidades

- Lista performática com 50 eventos gerados por mock.
- Visualização de palestras, workshops e hackathons.
- Favoritar eventos diretamente no card.
- Navegação por rotas nomeadas.
- Tela de detalhes com inscrição rápida.
- Validação de nome obrigatório e e-mail com `@`.
